DAMANG
============

Damang adalah aplikasi pendamping kesehatan dan olaharga. Damang memiliki fitur untuk selalu memantau kondisi kesehatan anda melalui data detak jantung, dan juga menjadi pendamping anda ketika anda berolahraga agar olahraga anda tenang dan aman.

### Special Pairing Procedures

Bebrapa device memerlukan key spesial untuk bisa pairing. Key tersebut bisa diperoleh melalui aplikasi official waerable device masing - masing. Untuk lebih jelasnya, silahkan buka link berikut: [Huami Server Pairing](https://codeberg.org/Freeyourgadget/Gadgetbridge/wiki/Huami-Server-Pairing)
